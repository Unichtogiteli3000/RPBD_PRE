# Клиентская часть музыкальной библиотеки

## Описание
Клиентская часть веб-приложения "Музыкальная библиотека" реализована с использованием HTML, CSS и JavaScript. Интерфейс полностью локализован на русский язык и соответствует требованиям проекта.

## Структура проекта
- `index.html` - Главная страница приложения с основным интерфейсом
- `login.html` - Страница входа в систему
- `register.html` - Страница регистрации нового пользователя
- `styles.css` - Файл стилей для всего приложения
- `script.js` - Основной файл с JavaScript-функциональностью

## Функциональность

### Основные разделы:
1. **Профиль пользователя**
   - Просмотр и редактирование личной информации
   - Смена аватара
   - Управление любимыми жанрами и исполнителями

2. **Треки**
   - Просмотр своих треков
   - Добавление новых треков
   - Редактирование и удаление треков
   - Отображение информации: название, исполнитель, жанр, BPM, длительность

3. **Коллекции**
   - Просмотр своих коллекций
   - Создание новых коллекций
   - Управление треками в коллекциях
   - Специальная коллекция "Любимые треки"

4. **Поиск**
   - Поиск треков по различным критериям (название, исполнитель, жанр, BPM, длительность)
   - Фильтрация результатов

5. **Админ-панель** (доступна только администраторам)
   - Управление пользователями
   - Просмотр всех треков
   - Просмотр журнала операций

### Особенности реализации:
- Все операции с базой данных выполняются через API-вызовы (в реальной системе будут вызовы хранимых процедур)
- Интерфейс адаптирован для разных размеров экранов
- Использованы модальные окна для форм ввода
- Реализована система навигации между разделами
- Проверка сессии пользователя

## Требования к серверной части
Для полноценной работы клиентской части необходима серверная часть, которая будет:
- Обрабатывать аутентификацию и авторизацию
- Предоставлять API-методы для всех операций с данными
- Выполнять вызовы хранимых процедур к базе данных
- Обеспечивать разграничение прав доступа (пользователь/администратор)

## Используемые технологии
- HTML5
- CSS3
- JavaScript (ES6+)
- Responsive Design
- Адаптивная верстка
